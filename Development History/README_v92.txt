BUILD: myos_v92_1_common_controls_mousewheel

Adds USER32 builtin common-control coverage for LISTBOX, COMBOBOX and SCROLLBAR on top of the v90/v91 binary DLGTEMPLATE dialog stack.

Highlights:
- registers builtin classes: LISTBOX, SCROLLBAR, COMBOBOX
- DLGTEMPLATE atom classes 0x0083/0x0084/0x0085 now create real controls
- LISTBOX: LB_ADDSTRING, LB_INSERTSTRING, LB_DELETESTRING, LB_RESETCONTENT, LB_GETCOUNT, LB_SETCURSEL, LB_GETCURSEL, LB_GETTEXT, LB_GETTEXTLEN, LB_FINDSTRING, LB_FINDSTRINGEXACT, LB_SELECTSTRING, selection state, mouse + keyboard navigation, LBN_SELCHANGE/SETFOCUS/KILLFOCUS
- SCROLLBAR: SBM_SETPOS, SBM_GETPOS, SBM_SETRANGE, SBM_GETRANGE, SBM_SETRANGEREDRAW, mouse arrows/page/thumb-track, keyboard navigation, WM_VSCROLL/WM_HSCROLL notifications
- COMBOBOX: CB_ADDSTRING, CB_INSERTSTRING, CB_DELETESTRING, CB_RESETCONTENT, CB_GETCOUNT, CB_GETCURSEL, CB_SETCURSEL, CB_GETLBTEXT, CB_GETLBTEXTLEN, CB_FINDSTRING, CB_FINDSTRINGEXACT, CB_SELECTSTRING, CB_SHOWDROPDOWN, CB_GETDROPPEDSTATE, dropdown hit-test/draw, CBN_* notifications
- DialogLab now has a ControlProbe dialog created from a real binary DLGTEMPLATE blob using LISTBOX/COMBOBOX/SCROLLBAR atoms.
