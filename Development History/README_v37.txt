myOS v37 - Controls v2

Adds ControlLab with USER32-style child HWND controls:
- BUTTON-lite with pressed state and WM_COMMAND/BN_CLICKED
- EDIT-lite with focus, text input, Backspace, Ctrl+C/Ctrl+V clipboard path
- LISTBOX-lite with Add Item, Clear, selection and WM_COMMAND/LBN_SELCHANGE

Start: Start menu -> ControlLab, optional F16.
